# Mailchimp Newsletter Signup

> Newsletter signup using Node.js/Express and the Mailchimp API. 

## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:4000
node app.js
```
### Environment Variables required:
* API Key
* List ID
* Mailchimp server number (usX)
